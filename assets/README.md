# 📁 Pasta Assets - Ícones e Splash Screen

Coloque suas imagens aqui para facilitar o processo de substituição.

## 📋 Arquivos Necessários

### Para o Ícone do App:
- **`icon.png`** - 1024x1024px (PNG com fundo transparente)
  - Este será usado para gerar todos os tamanhos necessários

### Para a Splash Screen:
- **`splash.png`** - 2732x2732px (PNG)
  - Para iOS (será redimensionado automaticamente)
  - Para Android, será usado como base para gerar os tamanhos

## 🎨 Especificações Recomendadas

### Ícone:
- **Tamanho**: 1024x1024 pixels
- **Formato**: PNG
- **Fundo**: Transparente (recomendado)
- **Conteúdo**: Deve ocupar ~80% do espaço (deixar margem de segurança)

### Splash Screen:
- **Tamanho**: 2732x2732 pixels (ou proporção 1:1)
- **Formato**: PNG
- **Conteúdo**: Pode ser a mesma imagem do ícone ou uma versão adaptada

## 📝 Próximos Passos

Após colocar os arquivos aqui:

1. Use uma ferramenta online para gerar os tamanhos:
   - **Android Asset Studio**: https://romannurik.github.io/AndroidAssetStudio/
   - **App Icon Generator**: https://www.appicon.co/

2. Ou siga o guia em `GUIA_ICONES.md` para substituição manual
